# TaiMountain

TaiMountain 取名泰山，意稳如泰山。

TaiMountain 是一个自动化运维运维平台系统。希望通过该系统的构建，可以帮助大家探索`DevOps`理念，争取在实际业务尽早落地。


## 主要技术栈

- python 3.6
- django 2.0+
- saltstack 2018.3.2

## 主要功能 

本系统主要功能模块：

- CMDB
    - 主机基本信息管理
    - 应用配置管理
- 发布管理 
- 任务管理
    - 定时任务
    - 触发任务
- 监控管理
- TODO 待补充

## 如何贡献

- 查看issue列表，尝试解决其中的问题。
- forking 仓库，创建一个新分支，编写新功能代码或修改现有代码bug。
- 修复后，编写测试，测试无误后，提交PR。
- 维护者确认后通过。 。
